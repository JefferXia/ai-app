# Aura 设计系统

## 产品背景

- **产品定位**：AI 情感陪伴应用，帮助用户放松入睡
- **目标用户**：都市独居青年、高压职场人、需要情感陪伴的人群
- **行业领域**：身心健康 / AI 陪伴 / 睡眠应用
- **项目类型**：对话式 Web 应用，沉浸式全屏体验

## 设计风格

- **风格方向**：温暖亲密感（Warm Intimacy）— 有机温度与精致质感的融合
- **装饰程度**：有意为之 — 微妙的渐变、柔和的光晕、有深度的氛围感
- **整体基调**：产品应像一个私密、温馨的空间，而非临床工具或自然纪录片
- **布局理念**：沉浸式优先 — 全屏角色背景主导体验，聊天 UI 以玻璃态叠加，不喧宾夺主

## 字体系统

- **展示/标题**：Instrument Serif — 温暖、优雅、有情感分量。衬线字体带来个性但不显老气
- **正文**：Geist（已在使用）— 简洁现代，与衬线字体搭配和谐
- **UI 标签**：同正文，保持一致性
- **数据/表格**：Geist Mono — 支持等宽数字，与现有配置一致
- **代码**：Geist Mono
- **加载方式**：Google Fonts CDN（Instrument Serif），本地托管（Geist）
- **字号比例**：12/14/16/18/24/32/48px（1.25 模数比）

## 颜色系统

- **色彩策略**：克制表达 — 主色用于强调，色彩稀有而有意义
- **主色**：#A78BFA（柔薰衣草紫）— 温暖但不粉嫩，与青色拉开差异
- **强调色**：#F59E0B（暖琥珀色）— 如烛光般亲密温暖
- **背景色**：#0F0A1A（深紫黑）— 比纯黑更温暖，有包裹感
- **表面色**：#1A1625（提升紫）— 卡片和叠加层使用
- **弱化文字**：#9CA3AF（冷灰）— 可读但柔和
- **语义色**：
  - 成功 #34D399
  - 警告 #FBBF24
  - 错误 #F87171
  - 信息 #60A5FA
- **深色模式**：已为深色优先设计，表面色在不同层级略有提升

## 间距系统

- **基础单位**：4px
- **密度**：舒适 — 这是一个亲密、放松的体验，不是数据仪表盘
- **比例**：2xs(2) xs(4) sm(8) md(16) lg(24) xl(32) 2xl(48) 3xl(64)

## 布局系统

- **布局方式**：沉浸式优先
- **网格**：单列为主，响应式适配
- **最大内容宽度**：无限制（全屏沉浸）
- **圆角**：分层级 — sm:6px, md:12px, lg:16px, full:9999px
- **玻璃态效果**：聊天界面叠加层使用 backdrop-blur + 半透明背景

## 动效系统

- **动效风格**：有意为之 — 柔和淡入、温和过渡、呼吸动画
- **缓动函数**：
  - 进入：ease-out
  - 退出：ease-in
  - 移动：ease-in-out
- **时长**：
  - 微交互：50-100ms
  - 短动画：150-250ms
  - 中等动画：250-400ms
  - 长动画：400-700ms

## 交互模式

### 语音输入（按住说话）

模仿微信的按住语音功能，提供更自然的语音交互方式。

**交互流程**：
1. 用户按下语音按钮 → 按钮变为录音状态（脉冲动画）
2. 录音中显示音波可视化 + 时长计时
3. 松手发送，或上滑取消（显示取消提示）
4. 发送后显示加载态 → 文字 + 音频回复

**UI 状态**：
- **默认态**：麦克风图标 + "按住说话" 文字提示
- **按下态**：按钮放大、背景变为主色、显示音波动画、时长计时
- **取消态**：上滑时显示 "松手取消" + 红色警示
- **发送态**：加载动画 → 消息气泡出现

**视觉规范**：
- 语音按钮位置：输入框左侧或底部固定
- 按下态尺寸：默认 56px → 按下 72px（平滑过渡）
- 音波动画：3-5 条波形，主色，脉冲频率 60bpm
- 取消区域：上滑超过 100px 触发取消提示

**触觉反馈**（移动端）：
- 按下：轻震动
- 开始录音：中等震动
- 取消：双震动
- 发送成功：轻震动
