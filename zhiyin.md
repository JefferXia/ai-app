# 产品需求文档 (PRD)：Aura (微光) —— AI 情感陪伴系统

**版本：** V2.0

**状态：** 已实现

**技术栈：** MiniMax TTS + M2-her LLM

**文档更新：** 2025-03-09

---

## 1. 项目背景与目标

### 1.1 痛点分析

- **情感性失眠：** 独居人群在深夜易产生孤独感，大脑因缺乏安全感而过度活跃。
- **反刍思维：** 睡前不由自主地反复思考白天的压力或过错，导致焦虑。
- **交互缺失：** 现有的冥想 App 缺乏反馈，用户感到是在面对冷冰冰的录音。

### 1.2 产品愿景

打造一个**具有深度共情能力**的 AI 陪伴角色，通过个性化角色对话，消除孤独感，为用户提供温暖的情感陪伴。

---

## 2. 用户角色 (User Personas)

- **都市独居青年：** 下班后社交清零，渴望深夜有人能听自己说话。
- **高压职场人：** 神经紧绷，需要非指令性的放松引导，而非生硬的冥想口令。
- **异地恋/异国学子：** 习惯了有人陪伴入睡，在独处时难以适应黑暗与安静。

---

## 3. 核心功能需求

### 3.1 角色人格系统

### 3.2 交互方式

#### **输入方式**
- 文本输入：用户通过输入框发送文字消息

#### **输出方式**
- 文本显示：AI 回复以聊天气泡形式展示
- 语音合成：AI 回复自动转换为语音播放
- 动作表情：括号内的动作描述（如 `(微笑)`、`(轻轻握住你的手)`）单独显示为斜体，不生成语音

### 3.3 角色切换

- **滑动切换：** 在页面任意位置左右滑动（>50px）切换角色
- **按钮切换：** 点击左右箭头按钮切换角色
- **圆点切换：** 点击底部角色指示点直接切换
- **背景切换：** 切换角色时背景图平滑过渡（500ms 动画）
- **对话清空：** 切换角色时自动清空当前对话历史，显示新角色的起始问候

### 3.4 背景音乐

- 用户可开启/关闭背景音乐（流水声）
- 音乐循环播放，音量 10%
- 与语音回复独立，不会相互干扰

## 4. 技术架构

### 4.1 AI 引擎

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| **LLM** | MiniMax M2-her | 支持角色扮演，多轮对话 |
| **TTS** | MiniMax speech-2.6-turbo | 高质量中文语音合成 |
| **音量** | vol: 5 | 语音音量 5 倍 |

### 4.2 角色背景图

```
public/
└── images/
    └── character/
        ├── wumei_yujie.jpg          # 妩媚御姐背景
        ├── female-shaonv-jingpin.jpg # 甜美少女背景
        └── bingjiao_didi.jpg        # 清澈邻家弟弟背景
```

### 4.3 API 架构

```
POST /api/aura/chat
├── 请求参数
│   ├── text: string        # 用户输入
│   ├── history: Message[]  # 对话历史
│   ├── voiceId: string     # 角色音色 ID
│   └── personality: string # 角色人格描述
├── 处理流程
│   ├── 1. LLM 生成文本响应
│   ├── 2. 提取动作/表情（括号内容）
│   ├── 3. TTS 合成语音（不含动作）
│   └── 4. 返回文本 + 音频
└── 响应
    ├── text: string       # 完整文本（含动作）
    ├── actions: string[]  # 提取的动作列表
    └── audio: string      # hex 编码的 MP3
```

## 5. 界面设计 (UI/UX)

### 5.1 视觉风格

- **全屏背景图：** 每个角色独立的背景图铺满整个界面
- **半透明遮罩：** 50% 黑色遮罩保证文字可读性
- **毛玻璃效果：** 消息气泡使用 backdrop-blur 效果
- **暗色调：** 紫色系主题，温暖柔和

### 5.2 布局结构

```
┌─────────────────────────────────┐
│         Aura (微光)              │  ← 标题
│         角色名称                  │
├─────────────────────────────────┤
│  ◀  [ 角色名称 ]  ▶              │  ← 切换按钮
│       ●  ○  ○                   │  ← 角色指示器
├─────────────────────────────────┤
│                                 │
│    [角色问候 / 对话消息]          │  ← 内容区
│                                 │
├─────────────────────────────────┤
│  ● 准备就绪           [音乐]     │  ← 状态栏
│  ┌───────────────────────┐      │
│  │ 输入框                 │ [发送]│  ← 输入区
│  └───────────────────────┘      │
│       Powered by AI              │
└─────────────────────────────────┘
```

### 5.3 消息气泡

- **用户消息：** 右对齐，紫色半透明背景
- **AI 消息：** 左对齐，白色半透明背景，带角色 emoji
- **动作表情：** 消息下方斜体灰色显示 `*动作*`
- **重播按钮：** AI 消息带重播按钮
